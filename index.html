a.Propose a rescaling strategy

Propose a recaling strategy(with enough details of the procedure) for the variable Year.Explain why the proposed strategy is appropriate

  • Two common methods:
    outilier: an outlier is a data point that differs significantly from other observations.
  1. Min-Max Scaling(Normalisation):
    $$
    X_{norm} = \frac{X - X_{min}}{X_{max} - X_{min}}
    $$
    This method scales the value between 0 and 1.(it is useful when the distriution of the data is not Guassian or when the algorithm you‚Äôre using relies on the input features being similar scale.)
  2. Standardization(Z-score Normalisation):
    $$
    X_{std} = \frac{X - \mu}{\sigma}
    $$
    This method standardizes the features by rescaling them to have a mean of zero and standard deviation of 1. Standardization assumes a Gaussian distribution of the data and is suitable when the features have different units or different features.
    if you want to be robust to outiliers, standardization could be a good choice.
  • If we use polynomial regression, the features are not just raw absolute values of the year; they also include higher-order terms.

  1. Rescaling can ensure features on a similar scale.
    Rescaling can ensure that all features, including those higher-order terms, are on a similar scale. Rescaling allows our model to still produce well-behaved values.

  2. Rescaling can ensure a faster convergence.

  3. Rescaling can prevent numerical issues.
    When optimising our parameters to enhance the performance of our model, we‚Äôre expected to manipulate many matrix operations, especially inversion operations. features of higher-order terms produced polynomial regressions can lead to numeric instability. Rescaling can ensure that the matrix values are within a resonable range, preventing numerical issues especially during the inversion operations.

numerical issues refer to ‚Äònumerical instability‚Äô, ‚Äòloss of precision‚Äô and computational efficiency

1.1 How graphs can be represented as matrices

The definition of a graph is G = (V, E), where V is a set of vertices and E is a set of edges connecting pairs of vertices.
given V set and E set, we can write the adjacency matrix A with each element Aij equal to the weight of edge from vi to vj(edge-weighted graph)

some helpful properties of graph matrix:

  1. in-degree(numbers of edges entering a vertex) and out-degree(numbers of edges leaving a vertex) of each vertex

  2. if the matrix is symmetric, it represents an undirected graph
    and we can get an undirected graph by calculating the sum of a directed graph and its transpose.

1.2 How flows on graphs can be represented as matrix operations

any matrix can be expressed by the product of three other matrices with special forms.
